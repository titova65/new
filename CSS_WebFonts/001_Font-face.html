В предыдущих версиях CSS разработчики были вынуждены использовать только те шрифты, которые гарантированно установлены на компьютере пользователя, в CSS3 разратотчики могут использовать любые шрифты, которые они захотят. Когда вы найдёте необходимый шрифт просто разместите его на веб-сервере и подключите его спомощью нового CSS3 свойства @font-face. Подключенный шрифт будет скачан и отображён автоматически при загрузке страницы.

Обратите внимание:Internet Explorer поддерживает шрифты в формате .еоt, а остальные браузеры в формате .ttf т.е. для того, чтобы шрифты отображались корректно во всех браузерах Вы должны подключить его к странице в двух форматах
